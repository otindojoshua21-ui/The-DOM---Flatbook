# Flatbooks DOM Lab

This project demonstrates DOM manipulation for the Flatbooks bookstore lab.

## What it does

- Selects the header element with `id="header"`
- Updates the header text to the `bookStore.name`
- Loops through each book in `bookStore.books`
- Creates `li`, `h3`, `p`, and `img` elements for each book
- Appends each book item into the `#book-list` container

## Run locally

Open `index.html` in a browser to see the bookstore catalog.
